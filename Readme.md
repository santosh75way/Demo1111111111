# Project Templates

This repository contains ready-to-use templates that help you start new projects faster. Instead of setting everything up from scratch, you can pick a template and build on top of it.

## What’s Inside?

```
auth-social-system/
auth-system/
react-setup/
.gitignore
Readme.md
```

### Auth System

A basic authentication template.
Use this when your project needs user login, registration, and protected access.

### uth Social System

An authentication setup with social login support.
Good for apps that allow users to sign in using social platforms.

### React Setup

A simple React project starter.
Perfect when you want a clean React setup without extra configuration work.

## How to Use

1. Clone this repository.
2. Open the folder of the template you want.
3. Copy it into your new project.
4. Customize it to fit your needs.

That’s it — no complicated setup.

## Why This Exists

* Saves time
* Keeps projects organized
* Avoids repetitive setup work

## Notes

- Each template is independent and can be used on its own.
- Feel free to modify or expand them as your project grows.
